# PHPstorm自定义快捷键

快捷键的定义是基于Netbeans6.5后， 使用Alt + 来配置常用快捷键，因为Alt键用左手的大拇指即可碰到，比用小拇指敲击Ctrl方便。

## 快捷键定义

- alt + o -> class search
- alt + p -> file search

- alt + e -> recent files
- alt + r -> file structure
- alt + t -> refactor this 

- alt + n -> new file 
- alt + , -> settings
- alt + ↑ -> Navigation Bar

- alt + shift + a -> enter action or option name
- alt + shift + o -> sylbol

- double shift

## 其它设置

- View -> Start Button 隐藏状态栏 
- View -> NavigationBar	隐藏导航
- View ->  Toolbar	隐藏工具栏
